# Immersive Cables
An unofficial port of Immersive Integration wires

This repository is a Minecraft 1.12.2 Cleanroom fork of Immersive Cables.
The original repository can be found [here](https://github.com/SanAndreaP/ImmersiveCables).

Cleanroom is a 1.12.2 Forge fork, providing newer toolchain, new APIs and 99% compatibility.

Need Java 25.

# Modification
It's fully compatible with [AE2S](https://github.com/FormlessDragon/Applied-Energistics-2-Supergiant) now.<br>
Additionally, the code related to the Rs part has been deleted.
